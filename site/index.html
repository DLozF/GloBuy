What it does

Everything you need.
Nothing you don't.

Full-page translation

Catches scroll-loaded listings, pop-ups, and filter panels — not just the header or first viewport.

Inline currency conversion

Every price converts to your local currency right where it sits, using today's live exchange rates.

Luxury glossary

Renders resale jargon like S급, N급, and BNWT correctly — not as literal, confusing translations.

Size conversion to US

Footwear and apparel sizing mapped automatically. 260mm ≈ US Men's 8. EU 38 ≈ US Women's 7.5. Always labeled.

Search in English

Type in English — the extension translates your query into the site's language so you find exactly what you want.

Private by default

Translation runs entirely on your device — nothing you read leaves your browser (only currency codes are sent). An optional cloud Premium tier is coming in a future release.

Private by default

Globuy uses Chrome's built-in on-device translation model. Page text is processed entirely within your browser — it never touches an external server. The only data that leaves your device is a currency code (e.g. "KRW") to fetch today's exchange rate. An optional Premium cloud mode is planned for a future release; it is not part of this version, and when it ships it will be off unless you turn it on.

Requires Google Chrome 138 or later. The on-device translation model downloads once on first use (~22 MB). Currency conversion is available on all Chrome versions.
